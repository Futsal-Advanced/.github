## 🏠 About Our Journey
> 단순히 코드를 짜는 것을 넘어, 사용자의 일상 속 페인 포인트(Pain Point)를 해결하는 최적의 솔루션을 설계하고 구축합니다. 

* **위고하이 (We Go High)**: 비즈니스 효율을 극대화하는 혁신적인 SaaS 플랫폼
* **rote**: 스포츠 현장의 생동감을 관리하는 정교한 타이머 서비스
